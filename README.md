# daniel-clickup-replicator

Webhook → ClickUp client onboarding. This service created a new Folder in a ClickUp Space when a form (Typeform, Jotform, Tally) was submitted, then replicated a predetermined layout of Lists and Tasks inside it. See `homebase` for the consolidated onboarding automation.
